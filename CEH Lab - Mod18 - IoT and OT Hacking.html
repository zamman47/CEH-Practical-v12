  • Machine-to-machine (M2M)/“Internet of Things” connectivity protocol
  • Useful for connections with remote locations where a small code footprint is required and/or network bandwidth is at a premium

Lab 2: Capture and Analyze IoT Device Traffic using Wireshark

MQTT is a lightweight messaging protocol that uses a publish/subscribe communication pattern.

  • Protocol is meant for devices with a low-bandwidth, so it is considered ideal for machine-to-machine (M2M) communication or IoT applications.
  • MQTT Broker web interface: Collects data and displays the status and messages of connected devices over the network

In Wireshark, simply filter to mqtt
